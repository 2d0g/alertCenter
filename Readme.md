# 安装要求
项目 | 要求
---- | ---
go    | 版本1.6.3及以上
beego | 版本1.7.0及以上

* 依赖gitlab账号体系，需要gitlab应用。
* 依赖mongodb服务作为存储。
# 功能说明
1. 兼容prometheus报警信息，比alertmanager更完善的功能
2. 多种安全验证方式的API
3. 支持报警信息分类存储，查询
4. 支持gitlab,ldap用户管理
5. 支持邮件，微信报警
6. 支持多维度报警忽略
7. 支持报警历史查询
8. 简洁完善的ui
